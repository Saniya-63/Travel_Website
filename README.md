✈️ Travel Website
A beautiful front-end travel website designed to inspire wanderlust and simplify trip planning. Explore destinations, check out packages, and get ready to book your next adventure—all from a clean, responsive interface.

🔗 Live Demo:
👉 Click to Explore  [ https://saniya-63.github.io/Travel_Website/]


💡 Project Highlights
🏠 Home Section
→ Eye-catching hero image with a call-to-action

📅 Booking Form
→ Enter destination, number of guests, and date to plan a trip

🧳 Packages Section
→ Attractive cards for featured travel packages

💼 Services Offered
→ Hotel booking, safety guidance, 24/7 support

🖼️ Gallery
→ Responsive image grid showcasing travel spots

👤 About Us
→ Project details with clean layout and aesthetic text design

🛠️ Tech Stack Used
HTML5 – Semantic layout for structure

CSS3 – Styling with responsive design and effects

JavaScript – Basic interactivity (form elements, menu toggle)
